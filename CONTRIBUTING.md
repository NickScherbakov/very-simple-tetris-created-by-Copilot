# Contributing to very-simple-tetris-created-by-Copilot

Thank you for your interest in contributing to this project! Your contributions are valuable and help make this project better.

## How to contribute

1. **Fork the repository**
   - Click the “Fork” button in the top right corner to create your own copy.

2. **Clone your fork**
   ```bash
   git clone https://github.com/YOUR_USERNAME/very-simple-tetris-created-by-Copilot.git
   cd very-simple-tetris-created-by-Copilot
   ```

3. **Create a new branch**
   ```bash
   git checkout -b feature/YOUR_FEATURE
   ```

4. **Make your changes**
   - Fix bugs, add features, or improve documentation.

5. **Test your changes**
   - Make sure your code works and does not break existing functionality.

6. **Create a Pull Request**
   - Push your branch to your fork and open a Pull Request to the main repository.
   - In the PR description, briefly explain what you changed and why.

## Guidelines

- Follow the code style used in the project.
- Make your commits meaningful and descriptive.
- If unsure, open an issue to discuss your idea before implementation.
- English is preferred for discussions, but Russian is also welcome if needed.

## Issue labels

- `good first issue` — tasks for newcomers, described in detail.
- `bug` — bugs and errors.
- `feature` — new features.

## Recognition

All contributors will be listed in the CONTRIBUTORS.md file.

Thank you for your contribution!  
If you have any questions, feel free to open an issue or contact the maintainer directly.
